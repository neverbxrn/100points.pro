Ультимативный гайд по установке скрипта

1

Установите Tampermonkey

Для начала работы вам нужен менеджер скриптов. Это расширение, которое позволяет запускать кастомные функции на сайтах.

Скачать для Chrome / Edge
!

Критически важная настройка

Без этого шага скрипт не будет работать из-за политики безопасности браузера (Chrome, Edge, Opera).

1. Перейдите в раздел расширений, вставив в адресную строку:
chrome://extensions

2. В правом верхнем углу включите тумблер "Режим разработчика" (Developer Mode).

3. Нажмите на иконку Tampermonkey в панели браузера -> Панель управления -> вкладка Настройки.

4. Пролистайте вниз до раздела "Безопасность" и убедитесь, что в пункте "Пользовательские скрипты" (Experimental Mode / User Scripts) стоит галочка или статус "Включено".

Внимание: Если вы используете Яндекс.Браузер, режим разработчика находится в разделе "Дополнения". Если тумблер не включен, Tampermonkey даже не предложит вам сохранить скрипт!
2

Скопируйте код скрипта

Нажмите на кнопку ниже, чтобы открыть файл с кодом. Скопируйте весь текст (Ctrl+A, Ctrl+C).

Открыть код скрипта
3

Создайте новый скрипт

Нажмите на иконку Tampermonkey в браузере -> "Создать новый скрипт". Удалите стандартный шаблон и вставьте скопированный код.

Затем нажмите Файл -> Сохранить (Ctrl+S).

4

Готово!

Просто обновите вкладку с личным кабинетом 100Points. Если всё сделано правильно, интерфейс преобразится автоматически.
