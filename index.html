Установка Tampermonkey

Для начала выберите расширение под ваш браузер. Оно необходимо для запуска скрипта.

Chrome / Edge / Opera Firefox
!

Критическая настройка (Важно)

Современные браузеры блокируют скрипты по умолчанию. Без этих действий ничего не заработает:

✓
Для Chrome, Edge, Yandex:
Введите в адресную строку chrome://extensions. В правом верхнем углу включите тумблер "Режим разработчика".
✓
Включение пользовательских скриптов:
1. Нажмите на иконку Tampermonkey в браузере -> Панель управления.
2. Перейдите во вкладку Настройки (Settings).
3. Измените "Режим конфигурации" на "Начинающий" или "Продвинутый".
4. Пролистайте вниз до раздела "Безопасность" и включите пункт "Пользовательские скрипты" (User Scripts).
2

Добавление кода

Теперь нужно перенести сам код скрипта в расширение:

  • Откройте файл с кодом: Открыть 100Points Pro.js
  • Нажмите Ctrl + A (выделить всё) и Ctrl + C (копировать).
  • В Tampermonkey нажмите "Создать новый скрипт" (значок плюса).
  • Удалите весь текст в редакторе и нажмите Ctrl + V (вставить).
  • Нажмите Файл -> Сохранить.
3

Проверка

Зайдите в личный кабинет 100Points. Скрипт должен активироваться автоматически. Вы увидите обновленные карточки уроков и боковое меню.
